# Online C++ Compiler(Powered by Wandbox and Paiza.IO)

## How to Build

### Requirements

- git
- Node.js v18.13.0

### Clone and setup

```
git clone https://github.com/mewmew-tea/WebCompiler.git
cd WebCompiler
npm install
```

### Run the application

```
npm run start
```

### Build executable

Windows(x64)

```
npx electron-builder --win --x64 --dir
```
